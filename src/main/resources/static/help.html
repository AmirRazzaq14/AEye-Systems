How do I get started?
  • Create an account on login.html
  • Complete profile-setup.html (age/gender/height/weight)
  • Use motion.html to track reps + calories burned
  • Use nutrition.html to log meals and view daily totals
Motion: camera permission is denied or not working
  • Make sure you clicked “Allow” for camera access in the browser prompt.
  • Check your browser’s site settings for this domain and enable Camera.
  • Close other apps/tabs using the camera, then retry.
  • For best tracking: stand back, good lighting, full body in frame.
Nutrition: why do I see meals I didn’t log?
If the Nutrition API can’t load your real log (auth/API error), the UI may show demo data. Check you’re logged in, and that today’s log exists (adding a meal will create it).
AI Coach: the coach won’t respond
The AI features require an API key. In deployment, set GEMINI_API_KEY. If the key is missing or rate-limited, you’ll see an error.
Known issues
  • Some features depend on backend configuration (Firebase credentials / AI keys).
  • Camera tracking quality varies based on lighting and device performance.
  • On first run, some pages may need a refresh after login depending on browser caching.
